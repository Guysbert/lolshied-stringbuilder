This is my first angularJS project. I bought a Lots Of Lights-Shield at 31c3. I wanted to extend the Fonts.cpp with missing characters.
To do that, you have to declare a string that defines which LEDs have to glow to display a certain character. Building that string and trying it on the arduino seemed a little bit tedious to me. To be honest I would've been long done now ;) But instead I created an angularJS script that does all the tedious work for me.
I guess I'll use this as an AngularJS Playground for some time.
If you'd lik to improve the webapp, you can fork the code on github. I'd love to see some improvement on the CSS.
